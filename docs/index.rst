elasticsearch-lua
=================

A simple low level client for elasticsearch written in lua. `elasticsearch-lua`_
is designed to be in accordance with other official clients for elasticsearch.

All API functions are developed to closely match with the REST API of elasticsearch.
The client is extensible and developers can attach their own plug-ins.
